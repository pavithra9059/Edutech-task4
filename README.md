A modern and responsive React.js single-page application featuring beautiful gradients, glassmorphism effects, animated cards, pricing sections, statistics, and a contact form.

✨ Features

- 🎨 Modern Glassmorphism Design
- 🌈 Gradient UI Elements
- 📱 Fully Responsive Layout
- ⚡ Fast React Components
- 📊 Statistics Dashboard
- 💰 Pricing Cards
- 📧 Contact Form
- 🚀 Smooth Hover Animations
- 🌙 Dark Theme Interface

🛠️ Technologies Used

- React.js
- CSS3
- JavaScript (ES6+)
